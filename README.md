# One Zero, Coffee ☕

An artisan coffee experience built with React, Vite, and Framer Motion. Featuring a premium frame-by-frame scroll animation system.

## ✨ Features

- **Premium Scroll Experience**: Custom `ScrollCanvas` component using GSAP ScrollTrigger for high-performance frame-by-frame coffee animations.
- **Dynamic UI**: Responsive landing page with elegant typography and fluid transitions using Framer Motion.
- **Micro-Animations**: Staggered text reveals, glassmorphism effects, and floating UI elements.
- **Artisan Aesthetic**: Carefully curated color palette and layout designed for a high-end coffee brand.

## 🚀 Tech Stack

- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: GSAP (ScrollTrigger) & Framer Motion
- **Icons**: Lucide React

## 🛠️ Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run dev server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
